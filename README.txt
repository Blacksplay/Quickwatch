App: Quickwatch - Die Recapp App



Entwickelt in:  			Android Studio 
Version: 				Android Studio Jellyfish | 2023.3.1 (Beta)

Zum Öffnen der App Empfohlen: 	Version Koala oder neuer

Die APK wurde auf einem Samsung S23 Ultra getestet

Umfang der Abgabe:

Die Abgabe umfasst drei vollständig implementierte Screens: den Filme-Tab, den Serien-Tab und die Detailseite für einen Film. Mit Ausnahme der Suchfunktion sind alle Features voll funktionsfähig. Die funktionierende Detailseite eines Films ist durch Klicken auf den obersten Film im Filme-Tab erreichbar.


Bildrechte:
